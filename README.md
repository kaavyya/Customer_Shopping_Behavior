# Customer_Shopping_Behavior
Data analytics visualization project showcasing customer behavior using Python, SQL and PowerBI
Customer Shopping Behavior Analysis
# Overview
This Data Analytics project focuses on analyzing customer shopping behavior using Python, MySQL, and Power BI. The objective is to clean and transform raw data, perform exploratory data analysis (EDA), extract business insights through SQL queries, and present findings through an interactive dashboard and professional reports.
The project demonstrates end-to-end data analytics skills, including data preprocessing, database management, visualization, and business reporting.

# Dataset : Customer Shopping Behavior Dataset

The dataset contains customer shopping information, including:
* Customer Demographics
* Gender  
* Age
* Purchase Amount
* Product Category
* Payment Method
* Loyalty Status
* Subscription Status
* Review Ratings
* Seasonal Purchases
* Purchase Frequency

# Tools & Technologies
* Programming & Analysis
* Python(Pandas, NumPy, Matplotlib ,Seaborn)
* Database : MySQL
* Data Visualization: Power BI
* Reporting : Microsoft PowerPoint,
Gamma AI

# Project Workflow
1. Data Loading : 
Imported dataset into Python using Pandas.
Examined data structure and column information.
2. Data Cleaning : 
Handled missing values.
Removed duplicates.
Renamed columns for consistency.
Corrected data types.
Treated outliers where necessary.
3. Exploratory Data Analysis (EDA) : 
Analyzed customer demographics.
Studied purchasing patterns.
Identified revenue trends.
Examined customer loyalty and subscription behavior.
Visualized key insights using charts and graphs.
4. SQL Analysis : 
Imported cleaned data into MySQL.
Created database and tables.
Executed SQL queries to:
Calculate total revenue.
Analyze revenue by gender.
Identify top-performing categories.
Evaluate customer purchasing behavior.
Generate business insights.
5. Dashboard Development : 
Built an interactive Power BI dashboard
6. Reporting & Presentation
Created a detailed project report summarizing findings.
Developed a presentation using Gamma AI.
Presented key business insights and recommendations.
